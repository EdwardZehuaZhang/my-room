## Codebase Patterns
- Use `npm install --include=dev` on this machine — global npm config has `omit=dev` set
- Client typecheck: `cd client && node_modules/.bin/tsc --noEmit` (npx tsc resolves to wrong package)
- Server typecheck: `cd server && node_modules/.bin/tsc --noEmit`
- CSS modules for component styles, globals.css for design system variables
- strict: true in both client and server tsconfig
- No external UI libraries — plain CSS modules only
- Zustand store at client/src/store.ts — usePlayerStore for player state
- Components live in client/src/components/ with co-located CSS modules
- Splat renderer: @lumaai/luma-web LumaSplatsThree — has onProgress/onLoad callbacks, is a Three.js Mesh (use <primitive>)
- App.tsx owns Canvas + Suspense boundary for the 3D scene; loading state managed via useState
